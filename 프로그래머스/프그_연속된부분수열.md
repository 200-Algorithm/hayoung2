# [프로그래머스] 연속된 부분수열의 합

1. 실패

```protobuf
def solution(sequence, k):
    answer = []
    n=len(sequence)
    
    for i in range(n):
        for j in range(n,0,-1):
            if sum(sequence[i:j]) ==k:
                if answer!=[]:
                    if answer[1]-answer[0] > (j+1)-(i):
                        answer=[i,j-1]
                    elif answer[1]-answer[0] ==  (j+1)-(i):
                        if answer[0]> i:
                            answer =[i,j-1]
                else:
                    answer=[i,j-1]

                
    return answer
```

1. 아 왜 생각해낼 수 있어 할수있어 ㅠ gpt 씀 근데 할 수 있엄

def solution(sequence, k):
    answer = []
    n=len(sequence)
    start,end=0,0
    result=sequence[0]
    min_len=float('inf')
    
    while True:
        if result ==k:
            current_len =end -start+1
            if current_len < min_len:
                min_len=current_len
                answer=[start,end]
                
            result-=sequence[start]
            start+=1
        elif result < k :
            end+=1
            if end>=n:
                break
            result+=sequence[end]
        else:
            result-=sequence[start]
            start+=1
        
        if start>=n:
            break         

                
    return answer
