# [프로그래머스] 피로도

1. 성공

```protobuf
from itertools import permutations
def solution(k, dungeons):
    answer = -1
    n=len(dungeons)
    tmp=k
    for dun in permutations(dungeons,n):
        cnt=0
        k=tmp
        for i in dun:
            if k >= i[0]:
                k-=i[1]
                cnt+=1
            else:
                break
        answer=max(cnt,answer)
        if answer==n:
            break
    return answer
```
