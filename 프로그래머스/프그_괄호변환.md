# [프그] 괄호변환

1. 괄호변환

```protobuf
def solution(p):
    answer = ''
    stack=[]
    for i in p:
        if i=='(':
            stack.append('(')
            answer+='('
        else:
            if stack==[]:
                answer+='('
                stack.append('(')
            else:
                stack.pop()
                answer+=')'

    if stack !=[]:
        answer +=')'*len(stack)
                
    return answer
```
