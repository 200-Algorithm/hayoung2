# [프로그래머스] 귤 고르기

1. 성공

```protobuf
from collections import defaultdict
def solution(k, tangerine):
    answer = 0
    arr=defaultdict(int)
    for i in tangerine:
        arr[str(i)] +=1
    
    arr=sorted(arr.items(),key=lambda x:x[1],reverse=True)
    
    for i,v in arr:
        if k > 0:
            k-=v
            answer+=1
        else:
            break
    return answer
```
