# [프그] 테이블 해시함수

1. 왜다틀림 ? 35.3점임 

```protobuf
def solution(data, col, row_begin, row_end):
    answer = 0
    
    data.sort(key=lambda x: (x[col-1], -x[0]))
    for i in range(row_begin-1,row_end):
        for j in data[i]:
            answer +=(j%(i+1))
    return answer
```

1. 마지막 xor 무시했음 완완

```protobuf
def solution(data, col, row_begin, row_end):
    answer = 0
    tmp=[]
    data.sort(key=lambda x: (x[col-1], -x[0]))
    for i in range(row_begin-1,row_end):
        answer=0
        for j in data[i]:
            answer +=(j%(i+1))
        tmp.append(answer)
    answer=tmp[0]
    for i in range(1,len(tmp)):
        answer =answer ^ tmp[i]
    return answer
```
