# [백준] 영단어 암기는 괴로워

1. 이건성공

```protobuf
from collections import defaultdict

a=defaultdict(list)
b=defaultdict(int)
n,m=map(int,input().split())
for i in range(n):
    ip=input()
    # 각 단어 몇 개인지 파악
    if len(ip) >= m:
        b[ip]+=1
        # 총몇개,길이,string
        a[ip]=[b[ip],len(ip),ip]
# 정렬 자주나오는 , 길이 , 알파벳순
a=sorted(a.values(),key=lambda x:(-x[0],-x[1],x[2]),reverse=True)
for i in a[::-1]:
    print(i[2])
```
