# [백준] 배열 복원하기

1. 이건성공

```protobuf
h,w,x,y=map(int,input().split())
a1=[[0]*w for _ in range(h)]
a2=[[0]*w for _ in range(h)]
b=[]
for i in range(h+x):
    tmp=list(map(int,input().split()))
    b.append(tmp) # 최종 입력 

for i in range(h):
    for j in range(w):
        a1[i][j]=b[i][j] # 앞부분 받음 

r=0
for i in range(x,h+x):
    c=0
    for j in range(y,w+y):
        a2[r][c]=b[i][j]
        c+=1
    r+=1
r=0
# x,y 떨어진거리 마이너스해서 값 구함 
for i in range(x,h):
    c=0
    for j in range(y,w):
        a1[i][j]=a1[i][j]-a1[r][c]
        c+=1
    r+=1

for i in a1: 
    print(*i)
```
