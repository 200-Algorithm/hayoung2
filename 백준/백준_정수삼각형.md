# [백준] 정수삼각형

1. 으아아아아아아아 

```python
n=int(input())
dp=[[0]*(i+1) for i in range(n)]
arr=[]
for i in range(n):
    arr.append(list(map(int,input().split())))
dp[0][0]=arr[0][0]
for i in range(1,n):
    dp[i][0]=dp[i-1][0] +arr[i][0] 
    dp[i][i]=dp[i-1][i-1] + arr[i][i]
    for j in range(1,i):# 중간은 둘 중에 더 큰 거 값 추가 
        dp[i][j]=max(dp[i-1][j-1],dp[i-1][j])+arr[i][j] 
print(max(dp[n-1]))
```
