# [백준] 등수구하기

1. 성공

```protobuf
n,m,p=map(int,input().split())
if n!=0:
  arr=list(map(int,input().split()))
  cnt=1
  # 숫자 같은데, 값이 적으면 -1  
  if n==p and m<=arr[-1]:
    print(-1)
  else:
	  # 값 꺼내서 m보다 작아지면 순위 나옴
    for i in arr: 
      if i>m:
        cnt+=1
      else: 
        print(cnt)
        break
    else:
        print(cnt)
else: # 0이면 1
  print(1)
```
