# [백준] 겹치는 건 싫어

```protobuf
n,k=map(int,input().split())
arr=list(map(int,input().split()))
tmp=[0]* 100001
start=0
ans=0
for end in range(n):
    tmp[arr[end]] +=1
    while tmp[arr[end]]>k:
        tmp[arr[start]]-=1
        start+=1
    cnt=end-start+1
    if ans<cnt:
        ans=cnt
print(ans)
```
