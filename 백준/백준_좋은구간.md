# [백준]  좋은 구간

1. 성공

```protobuf
from itertools import combinations
l=int(input())
arr=list(map(int,input().split()))
n=int(input())

arr.sort()
minv,maxv=arr[0],arr[-1]
right,left=0,1
answer =[]
while left < l :
    for i in range(arr[right]+1,arr[left]):
        for j in range(i,arr[left]):
            if i <= n <= j and i<j :
                answer.append([i,j])
    right+=1
    left+=1

for i in range(1,arr[0]):
    for j in range(i,arr[0]):
        if i<=n<=j and i<j:
            answer.append([i,j])

print(len(answer))
```
