# [백준] 예산

```protobuf
n=int(input())
arr=list(map(int,input().split()))
money=int(input())
arr.sort()
left=1
right=arr[-1]
ans=0
while left<=right:
    mid=(left+right)//2
    tmp=0
    for i in arr:
        if i<mid:
            tmp+=i
        else:
            tmp+=mid
    if tmp <=money:
        ans=mid
        left=mid+1
    else:
        right=mid-1
print(ans)

```
