# [백준]  스킬 트리

1. 성공

```protobuf

def solution(skill, skill_trees):
    answer = 0
    m=list(skill)
    for sk in skill_trees:
        check=[]
        for s in sk:
            if s in m:
                check.append(s)
        ch=len(check)
        if check==m[:ch]:
            answer+=1

            
    return answer
```
