# [백준] 삼각형과 세 변

1. 성공

```python
import sys
input=sys.stdin.readline
x,y,z=map(int,input().split())
while x!=0 and y!=0 and x!=0:
    cnt=0
    if x==y:
        cnt+=1
    if x==z:
        cnt+=1
    if z==y:
        cnt+=1
    maxv=max(x,y,z)
    total=x+y+z-maxv

    if maxv < total and cnt==3:
        print('Equilateral')
    elif maxv < total and cnt==1:
        print('Isosceles')
    elif maxv < total:
        print('Scalene')
    else:
        print('Invalid')
    x, y, z = map(int, input().split())

```
