# [백준] 카약과 강풍

1. 성공

```protobuf
n,s,r=map(int,input().split())
arr1=list(map(int,input().split()))
arr2=list(map(int,input().split()))
cnt=0
tmp=[]
for i in arr2:
    if i not in arr1 :
        if i!=0 and i-1  in arr1 and i-1 not in tmp:
            tmp.append(i-1)
            continue
        if i+1 in arr1 and i+1 not in tmp:
            tmp.append(i+1)
            continue
    else:
        if i not in tmp:
            tmp.append(i)

print(len(arr1)-len(tmp))
```
